Welcome to the integrative programming practical work 2

This project has the following objectives:

  • To integrate the knowledge and practical skills acquired in the course units.
  • To reinforce the skills necessary to solve a problematic situation using object-oriented programming with C#, incorporating advanced OOP and client-server application concepts through the development of a Web API.
  • To model a data abstraction for the proposed domain by applying design patterns and good programming practices.
  • To design and integrate a web interface using HTML, CSS, and JavaScript development technologies that consumes a service layer published as a Web API.
